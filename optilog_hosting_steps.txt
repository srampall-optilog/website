
Website Hosting Steps for optilog.in (GitHub Pages + GoDaddy)

PART 1: GitHub Setup

1. Go to https://github.com
2. Click New Repository
3. Enter name: optilog.in
4. Select Public
5. Click Create Repository

6. Upload your website files:
   - index.html (required)
   - css/
   - js/
   - images/

7. Enable GitHub Pages:
   - Open repository
   - Click Settings → Pages
   - Select Source: Deploy from branch
   - Select Branch: main
   - Select Folder: / (root)
   - Click Save

8. Add custom domain:
   - In Pages settings
   - Enter: optilog.in
   - Click Save


PART 2: GoDaddy DNS Setup

9. Login to GoDaddy
10. Click My Products
11. Select domain: optilog.in
12. Click DNS

13. Add these 4 A records:

Type: A   Name: @   Value: 185.199.108.153
Type: A   Name: @   Value: 185.199.109.153
Type: A   Name: @   Value: 185.199.110.153
Type: A   Name: @   Value: 185.199.111.153

14. Add CNAME record:

Type: CNAME   Name: www   Value: srampall-optilog.github.io.


PART 3: Final Step

15. Wait 5–10 minutes

16. Open in browser:
    https://optilog.in

Your website is now live.

Future updates:
- Upload new files to GitHub repository
- Changes will automatically update the website
